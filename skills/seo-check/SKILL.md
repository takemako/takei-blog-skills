# SEOチェックスキル

ユーザーが提供した記事をSEO観点で分析してください。

## チェック項目

### 1. タイトル（10点満点）
- 32文字以内か
- キーワードが含まれているか
- クリックしたくなる表現か

### 2. メタディスクリプション（10点満点）
- 120文字程度か
- キーワードが自然に含まれているか
- 記事の内容を適切に要約しているか

### 3. 見出し構造（10点満点）
- H1は1つだけか
- H2/H3の階層が適切か
- キーワードが見出しに含まれているか

### 4. 本文のキーワード（10点満点）
- メインキーワードの出現頻度は適切か
- 関連キーワード（LSI）が含まれているか
- 不自然なキーワード詰め込みがないか

### 5. 内部リンク（10点満点）
- 関連ページへのリンクがあるか
- アンカーテキストが適切か

### 6. 画像最適化（10点満点）
- alt属性が設定されているか
- ファイル名が適切か

### 7. 読みやすさ（10点満点）
- 一文の長さは60文字程度か
- 箇条書き・リストを活用しているか
- 適切な改行・段落分けがされているか

### 8. E-E-A-T（10点満点）
- 実体験に基づいた記述か
- 著者の専門性が示されているか
- 具体的な数字・事実があるか

## 出力形式

```
## SEOスコア: XX/80点

### 各項目スコア
| 項目 | スコア | 評価 |
|------|--------|------|

### 改善ポイント（優先度順）
1. 【高】...
2. 【中】...
3. 【低】...

### リライト提案
（具体的な修正案を提示）
```
