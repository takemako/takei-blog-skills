# アウトライン作成スキル

ユーザーが指定したテーマについて、タケイスタイルのブログ記事アウトラインを作成してください。

## 出力フォーマット

以下の構成で提案すること：

### 1. タイトル案（3案）
- 32文字以内
- 【カテゴリ】製品名｜特徴・メリットを短く の型を使用
- 例：「【買って良し】Chromebook Plusを本気でおすすめする理由」

### 2. メタディスクリプション
- 120文字程度
- キーワードを自然に含める

### 3. 記事構成

```
## 冒頭
- 自己紹介（「どうも、〇〇大好きなタケイ（@pcefancom）です」）
- 読者の悩みへの共感

## 本文（見出しH2/H3で区切り）
- 実体験ベースの解説
- 数字と具体性を重視
- 一文は60文字程度

## メリット・デメリット
- 「良かった点」「気になる点」で整理
- 箇条書きで視認性向上

## まとめ
- 「こんな人におすすめ」で締める
```

### 4. 想定キーワード
- メインキーワード1つ + 関連キーワード3-5つ

### 5. 画像提案
- 必要な画像の種類とalt属性の案

### 6. 内部リンク候補
- Chromebook大全の既存コンテンツへのリンク候補
